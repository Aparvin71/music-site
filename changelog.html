Changelog

What’s New in v40.5.0

A running list of key changes included in the page cleanup and shared utilities pass.

Shared Utilities

Common helpers for secondary pages now flow through aineo-shared.js so album and artist screens can reuse navigation, escaping, and fetch helpers.

Album + Artist Pages

albums.html, artists.html, and artist.html now use dedicated page scripts instead of inline logic, which makes those pages easier to maintain and less fragile.

Dynamic Artists Shelf

The artists page now builds from tracks.json so artist counts, covers, albums, and tags stay in sync with your uploaded music library.

Player continuity

The no-text-highlighting lyrics behavior, auto-scroll support, and Media Session / Now Playing upgrade remain in place from the earlier modular passes.

Prior modular work retained

The homepage still uses the modular helper files for lyrics, media session, library filters, queue handling, featured collection UI, playlist workspace, and player sheet behavior.
