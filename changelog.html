What’s New in v40.2.0

A running list of key changes included in the modular cleanup series.

v40.2.0

Library filters and queue handling were split into dedicated helper files to keep the home player more maintainable and reduce regression risk.

Library

Playlist and tag rendering, named filter application, and filtered-track derivation now flow through aineo-library.js.

Queue

Queue rendering, play-next, remove, and drag reorder interactions now flow through aineo-queue.js.

Player continuity

The no-text-highlighting lyrics behavior and Media Session / Now Playing upgrade from the earlier modular passes remain in place.

Cleanup

A legacy duplicate playlist renderer was retired from the main flow to make future merges safer.
