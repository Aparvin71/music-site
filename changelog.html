What’s New in v39.8.0

A running list of key changes included in the current cleaned and stabilized package.

v39.8.0

Stability release with auto-scroll lyrics repair, Media Session / Now Playing support, queue cleanup, and version consistency across the site.

Lyrics

Auto-scroll now works with synced LRC files when present and falls back to estimated scrolling from inline lyrics when a track has no external lyrics file.

Now Playing

Added Media Session metadata, lock-screen / headset / vehicle remote action support where the browser allows it, plus playback position updates.

Cleanup

Fixed a malformed queue section attribute, restored consistent “Request a Song” navigation labels, and refreshed app version references.

Library

Featured Collection behavior, favorites, recently played, album pages, playlist tools, offline helpers, and queue tools remain in place.
