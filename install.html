Install

Get the App

Install Aineo Music on your phone, tablet, or computer for a cleaner app experience, quick launching, and stronger offline support.

Back to Music

iPhone / iPad

Install from Safari

  1. Open Aineo Music in Safari.
  2. Tap the Share button.
  3. Select Add to Home Screen.
  4. Tap Add.

Android

Install from Chrome

  1. Open Aineo Music in Chrome.
  2. Tap the menu in the browser.
  3. Choose Install app or Add to Home screen.
  4. Confirm the install.

Desktop

Install in Chrome or Edge

  1. Open the site in Chrome or Edge.
  2. Look for the install icon in the address bar.
  3. Click Install.

Why Install?

Better App Feel

  • Launch from your home screen.
  • Use a more focused, app-like layout.
  • Keep key files ready for offline use.
  • Get cleaner fullscreen playback on mobile devices.
